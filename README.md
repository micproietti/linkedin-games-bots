## LinkedIn Solver Bots
A lightweight Google Chrome extension that automatically solves the daily LinkedIn games

## Features
* **One-Click Solve:** Instantly starts solving the puzzle right before your eyes.
* **Emergency Stop:** A "Stop Solving" button in case you want to take over manually.
* **Runs Locally:** No account credentials are required. It uses your active LinkedIn session securely.

## How to Install (Developer Mode)
Installing the extension:
1. Download the folder containing the bot you want to try as a ZIP
2. Open Google Chrome and go to `chrome://extensions/` in your address bar
3. Turn on **Developer mode** using the toggle switch in the top right corner
4. Click the **Load unpacked** button in the top left
5. Select the extracted folder
6. The extension is now installed! Pin it to your toolbar for easy access

## How to Use
1. Go to the [LinkedIn Game](https://www.linkedin.com/games/) page
2. Select the game you want to use the bot on
3. Wait for the game board to fully load
4. Click the extension icon in your Chrome toolbar
5. Click **Solve It!**
